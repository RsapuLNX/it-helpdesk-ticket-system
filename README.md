\# IT Help Desk Ticketing System



Project Overview

This is a basic IT Help Desk Ticketing System built using Python and Flask.

It simulates a real-world IT support workflow where users can submit IT issues

and support staff can view and track tickets.



Technologies Used

\- Python

\- Flask

\- SQLite

\- HTML



Features

\- Submit IT support tickets

\- Set ticket priority (Low / Medium / High)

\- Store tickets in database

\- Admin panel to view all tickets



How to Run This Project

1\. Install Python

2\. Install Flask:

   pip install flask

3\. Run database setup:

   python database.py

4\. Start the app:

   python app.py

5\. Open browser:

   http://127.0.0.1:5000



Use Case

This project is designed for IT Support / Help Desk roles to demonstrate

basic backend skills, ticket management, and troubleshooting workflow.
